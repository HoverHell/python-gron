# How to contribute

We welcome contributions! Just fork the repository, clone it and start
hacking:

```bash
$ git clone git@github.com:YOU/python-gron.git
$ cd python-gron
$ # create a virtualenv to work in
$ python3 -m venv venv
$ . venv/bin/activate
$ pip install . -e['dev']
$ # create a branch and add features/fix bugs
$ pytest
$ git push
$ # make pull request
```
